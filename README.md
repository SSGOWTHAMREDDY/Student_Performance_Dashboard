Student Performance Dashboard

Project Overview
The Student Performance Dashboard is a data analytics project designed to analyze and visualize student academic performance. This dashboard helps identify top-performing students, subject-wise performance trends, and the impact of attendance on marks.
The project is built using Power BI to provide interactive and insightful visualizations for better decision-making.

Objectives
Analyze student performance across different subjects
Identify top and low-performing students
Evaluate the impact of attendance on marks
Compare performance based on gender and class
Provide an interactive dashboard for easy insights

Dataset Description
The dataset used in this project contains the following fields:
Student Name – Name of the student
Class – Class/grade of the student
Subject – Subject name
Marks – Marks obtained
Attendance – Attendance percentage
Gender – Male/Female
Exam Date – Date of examination
Teacher – Subject teacher

Dashboard Features
KPI Cards (Average Marks, Total Students, Attendance %)
Bar Chart – Subject-wise Marks Distribution
Line Chart – Performance Trend Over Time
Top Performers Identification
Gender-wise Performance Comparison
Attendance vs Marks Analysis
Interactive Slicers (Class, Subject, Gender, Date)

Tools & Technologies Used
Power BI – Data visualization
Microsoft Excel – Data preparation
DAX (Data Analysis Expressions) – Calculations and measures

Process / Methodology
Data collection and cleaning using Excel
Data import into Power BI
Data transformation using Power Query
Creating relationships between tables (if applicable)
Designing visualizations and KPIs
Adding slicers and filters for interactivity
Final dashboard layout and formatting

How to Use
Download the .pbix file from this repository
Open it using Power BI Desktop
Use slicers to filter data and explore insights

Key Insights
Students with higher attendance tend to score better
Certain subjects have consistently lower scores
Top performers can be identified for recognition
Gender-based comparison reveals performance patterns
